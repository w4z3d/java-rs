# java-rs

**java-rs** is a Rust library for reading and writing Java bytecode.  
It provides low-level primitives to parse, inspect, modify, and emit `.class` files.

## Features
- Java `.class` file parsing
- Bytecode inspection and manipulation

## Build

### Prerequisites
- Rust (install via [rustup](https://rustup.rs))

### Steps
```bash
git clone https://github.com/w4z3d/java-rs.git
cd java-rs
cargo build --release
```
